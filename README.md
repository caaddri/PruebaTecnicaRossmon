# Uso del Sistema

## 1. Registro
El usuario debe registrarse en la plataforma.

## 2. Inicio de sesión
Una vez registrado, debe ingresar al sistema.

## 3. Gestión de productos
Dentro del sistema, el usuario podrá:

- Crear productos
- Editar productos
- Eliminar productos

- Nota: ProductosBD es la base de datos es un .bak
